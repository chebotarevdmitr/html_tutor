Хотя символами /* */ можно оформлять и однострочный комментарий, однако в С++ для этого используется специальная пара символов ’/’ (//), указывающая начало строки комментария.
В этом случае концом комментария считается конец строки, так что нет необходимости отмечать его специальным символом. Этот способ наиболее полезен для коротких комментариев и является наиболее распространенным.

Программа – это последовательность команд (инструкций), которые помещаются в памяти и выполняются процессором в указанном порядке. Программа записывается на языке высокого уровня, наиболее удобном для реализации алгоритма решения определенного класса зада

В примере служебное слово int (выделено фиолетовым) перед main() означает, что главная функция должна вернуть операционной системе целочисленный код завершения программы с помощью оператора return (расположен ниже и также выделен фиолетовым). Возвращаемое значение нуль в операторе return обозначает «по умолчанию» нормальное завершение программы.
