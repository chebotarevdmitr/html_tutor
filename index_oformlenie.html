теги для оформления

Strong - выделяет текст жирным нначертанием . Визуальный эффект ,имеет смысловое
(семантическое) значение текст выделенный таким тегом важен по смыслу ,
при чтении на нём стоит сделать акцент,подчеркнув его значимость

Bold - выделяет текст жирным начертанием
имеет только визуальную нагрузку

Em - Emotional. выделяет текст курсивом
имеет смысловое значение

I - Italik выделяет текст курсивом имеет только визуальную нагрузку

U - underline тег для подчеркивания текста устарел в HTML5
вместонего использовать span с соответствующим классом

S -Stroke делает текст перечёркнутым

Sup -верхний индекс Отображает текст в виде верхнего индекса . Например 5 2

Sub -нижний индекс Отображает текст в виде нижнего индекса . Например H 2O

big -увеличивает шрифт текста на 1 по сравнению собычным

Small уменьшает шрифт текста на 1 по сравнению с обычным

code -используется для обозначения кода #include

q -Quote тег для выделения цитаты внутри абзаца. Цитата внутри текста : Hello Dima

blocquote -тег для выделения отдельно стоящей цитаты
